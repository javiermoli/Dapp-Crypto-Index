# Dapp-Crypto-Index

- This Dapp recreates an index of certain tokens as NFT.
- Once the NFT is burned, the user can choose to receive the entire amount of tokens or the amount in DAI.
- Only the Owner of the contract can mint an NFT.
- Only the owner of the NFT can burn it.
- The nft can be sold on the market.

Only the Kovan test net is supported.
